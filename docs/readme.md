# README.md

# 《赛博放生》Cyber Release

> 一个一本正经但越来越离谱的互联网生态行为艺术平台。

------

# 一、项目简介

《赛博放生》是一个基于 GIS 地图可视化 的互联网娱乐项目。

用户可以：

- 在地图上选择水域
- 选择想要放生的鱼类
- 点击“赛博放生”
- 对全球电子生态造成影响
- 获得电子功德

系统会：

- 自动生成生态事件
- 实时广播用户行为
- 演化数字生态系统
- 生成电子生态报告

项目核心：

# “一本正经地胡说八道”

------

# 二、产品定位

本项目不是：

❌ 环保平台
❌ 企业系统
❌ 真正的放生工具

而是：

# 互联网行为艺术 + 地图互动娱乐产品
